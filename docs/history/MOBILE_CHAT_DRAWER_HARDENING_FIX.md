# Mobile Chat Drawer Hardening Fix

Fixed mobile chat overlay/drawer issues:

- Chat shell no longer paints a dark full-screen layer over the page.
- Dark mode no longer hides page contents when chat is closed/opening.
- Chat panel opens as a visible mobile bottom sheet.
- Close button is larger, fixed in the chat header, and easier to tap.
- Launcher hides while chat is open on mobile.
- Mobile chat height, scrolling, emoji bar, and composer are contained.

No business logic changed.
