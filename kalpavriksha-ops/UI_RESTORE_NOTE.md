# UI Restore Note

The previous polished Kalpvriksha Designs Ops UI has been restored for both run modes:

- Root run: `npm run dev`
- Frontend run: `cd frontend && npm run dev`

The older simplified `frontend/src/style.css` UI has been removed. The frontend now uses the main polished React/Tailwind UI from `src/App.jsx`.

Backend PostgreSQL/email configuration remains in `backend/`.
