# Manager User Manual

## Main Areas
- Command Centre: assigned work, review queue and team status.
- Operations: assign and monitor active cases.
- My Tasks: personal assigned work.
- Attendance: team availability.
- Chat and Notifications: coordination.

## Daily Manager Workflow
1. Open Command Centre.
2. Check unassigned and under-drafting cases.
3. Assign cases to available designers.
4. Review completed uploads.
5. Send back revision if required.
6. Approve internal review when work is correct.
7. Monitor SLA alerts.
