Email OTP status:
- The current error shown in the screenshot is not a React/frontend bug.
- Gmail is rejecting the SMTP login: 535-5.7.8 Username and Password not accepted.
- This usually means SMTP_PASS/EMAIL_PASS is a normal Gmail password instead of a Google App Password, or the Gmail account does not allow that SMTP login.
- Required backend .env values for Gmail:
  EMAIL_PROVIDER=gmail
  SMTP_HOST=smtp.gmail.com
  SMTP_PORT=465
  SMTP_SECURE=true
  SMTP_USER=your Gmail address
  SMTP_PASS=your 16-character Google App Password
  OTP_FROM_EMAIL=the same Gmail address
