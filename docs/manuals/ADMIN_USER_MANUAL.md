# Admin User Manual

## Main Areas
- Command Centre: live operational overview.
- Operations: active case handling.
- Ledger: payments and finance tracking.
- Reports: workload, productivity, SLA and payment analytics.
- Production QA: readiness checklist and export.
- Attendance: team presence and break review.
- Archive: completed case history.

## Daily Admin Workflow
1. Open Command Centre.
2. Check waiting assignments, SLA violations, revisions and payment pending queues.
3. Assign urgent cases.
4. Review internal review queue.
5. Check pending payments in Ledger.
6. Export reports when required.
7. Review Production QA before release/deployment changes.

## Finance Access
Finance and ledger sections are Admin-only. Managers and Designers should not see finance summaries.
