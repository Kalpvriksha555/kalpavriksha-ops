# Designer User Manual

## Main Areas
- Command Centre: personal task overview.
- My Tasks: assigned drafting and revision tasks.
- Chat and Notifications: team communication.
- Profile: personal details and password changes.

## Daily Designer Workflow
1. Login and confirm availability.
2. Open My Tasks.
3. Start assigned task.
4. Upload completed file.
5. Respond quickly to revision requests.
6. Use break status when unavailable.
7. Logout at the end of the workday.
