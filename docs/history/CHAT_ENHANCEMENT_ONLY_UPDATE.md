# Chat Enhancement Only Update

This update is intentionally limited to the chat module.

## Improved
- Added a star-style unread indicator on direct-message users who have unread individual messages.
- The star remains until the receiver opens that person's chat.
- Added quick emoji picker in the chat composer.
- Improved attachment cards with cleaner file preview, type label, file size, image/video preview, and download action.
- Polished chat composer controls without changing other modules.

## Not touched
- Attendance
- Team availability
- Team activity
- Admin controls
- Cases
- Finance
- Dashboard
- Authentication
- Email/OTP
- Database logic

## Validation
- Production frontend build completed successfully.
