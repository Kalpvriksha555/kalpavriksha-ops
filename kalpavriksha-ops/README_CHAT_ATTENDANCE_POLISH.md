# Chat, Attendance, Availability and Notification Polish

This update keeps the existing UI and workflow, and applies the requested corrections:

- Khushbu Pandey name and username normalized globally; username is `khushbu`.
- Attendance excludes admin records.
- Command Centre Team Availability hides admins for non-admin users; admins can see other online admins and team members, but not themselves.
- Team Availability tabs show Available, Busy, Break, and Offline members separately.
- Chat now shows green online/available status in direct messages.
- Messages now include WhatsApp-style sent/read ticks.
- Global chat read receipts track who has read a message and when.
- Notifications are sorted newest-first and remain categorized by task, urgent, and mention.

Build checked with `npm run build`.
