# KalpaVriksha Ops - Recommended Features Upgrade

This upgrade keeps the existing look and workflow while adding/refining:

- Command Centre fixed and added as the default landing page.
- Banker performance moved to Admin-only Insights.
- Designer Leaderboard visible to everyone.
- Team workload overview now counts the current month only, so every month starts fresh while history remains stored.
- Team & Performance and Attendance Log are visible to everyone; admin-only employee controls remain protected.
- Added Team Availability, Urgent Revision Queue, Latest Activity Feed, and today's operations metrics.
- Added Admin Insights for monthly banker/bank performance, received amount, and pending collection.

Build verified with: npm run build
