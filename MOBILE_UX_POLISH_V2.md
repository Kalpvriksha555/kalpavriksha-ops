# Mobile UX Polish v2

Scope: mobile readability, navigation, chat, notifications, touch targets, tables, filters, and modal ergonomics only.

No business logic, API calls, roles, task workflow, attendance calculations, chat payloads, or backend code were changed.

Highlights:
- More reliable mobile tab tapping.
- Compact sticky header and tab rail.
- Better spacing and typography on phone screens.
- Tables scroll horizontally instead of breaking layout.
- Chat uses a single bottom-sheet style panel on mobile.
- Floating chat preview stays hidden on mobile to avoid duplicate chat surfaces.
- Notification panel/toasts stay above page content and within viewport.
- Badge numbers have enough space.
- Modals fit mobile viewport safely.
- Buttons and inputs have larger touch targets.
