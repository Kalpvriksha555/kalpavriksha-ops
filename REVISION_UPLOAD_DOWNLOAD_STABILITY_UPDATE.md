# Revision + File Transfer Stability Update

## Fixed

- Revisions raised on an assigned case now remain visible in the assigned user's My Tasks immediately.
- Temporary revision work items are explicitly flagged for My Tasks visibility.
- My Tasks now keeps active revision work visible regardless of the selected operations date.
- Uploads now show a clear live progress panel with filename, percentage, and status.
- Users are warned not to upload the same file again while an upload is already in progress.
- Upload complete and upload failed states are shown clearly.
- Download actions now show a visible “preparing/download started” status so users know the click worked.
- Revision/note attachment uploads now also show progress.
